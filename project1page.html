Employee Management System

This project provides an Employee Management System with a graphical interface developed in Java and JavaFX.

Project Overview

This system manages employee records, offering functionalities to add, update, remove, and display information. It uses a linked list structure for managing employee data and ensures that records are efficiently accessible and modifiable through a GUI interface.

Core Components

1. Node.java

Defines the Node class to represent an employee. Each node stores data attributes such as id, name, address, salary, phone, and work hours, along with a link to the next node.

2. LinkedList.java

Implements a singly linked list with methods for:

  • Add: Insert an employee record at the front of the list.
  • Show: Display employee information by ID.
  • Update: Adjust employee salary based on their work hours.
  • Remove: Delete an employee record by ID.

3. HelloController.java

Acts as the controller to connect the GUI with the linked list functionalities. It handles user actions such as:

  • Add Employee: Adds a new employee based on user input.
  • Update Salary: Updates an employee's salary if the work hours meet specified requirements.
  • Remove Employee: Removes an employee by their ID.
  • Show Employee: Displays details of a selected employee.

4. hello-view.fxml

This file structures the graphical user interface layout using JavaFX. It provides input fields, buttons, and tabs for different functionalities (e.g., Add, Update, Show).

5. HelloApplication.java

The main application file that launches the JavaFX application and sets up the primary window with a specified title and dimensions.

Features

  • User-friendly GUI developed with JavaFX.
  • Efficient linked list structure to manage employee data.
  • Validation of inputs for error handling and data integrity.
  • Easy-to-navigate tabs for adding, updating, removing, and displaying employee details.

How It Works

Upon launching the application, users can interact with the system through a series of tabs, each serving a specific function (e.g., adding a record, updating salary, etc.).
